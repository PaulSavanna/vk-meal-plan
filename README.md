# 🍽 VK Meal Planner Bot

Бот для планирования меню на неделю со списком покупок.

## Возможности

- 📖 Добавление рецептов с ингредиентами
- 📋 Планирование завтраков, обедов, ужинов
- 🛒 Автогенерация списка покупок
- 📅 План на всю неделю

## Установка

```bash
pip install -r requirements.txt
cp .env.example .env
python -m bot.main
```

## Как использовать

1. Добавь рецепты (название + ингредиенты)
2. Распланируй неделю
3. Получи список покупок

## Команды

- `/start` — главное меню
- `/help` — помощь
