# VK Meal Planner Bot

A VK bot for weekly meal planning with auto-generated shopping lists.

## Features

- Add recipes with ingredients
- Plan breakfasts, lunches, and dinners
- Auto-generate shopping lists
- Full weekly plan overview

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
python -m bot.main
```

## Usage

1. Add recipes (name + ingredients)
2. Plan your week
3. Get a shopping list

## Commands

- `/start` - main menu
- `/help` - help
